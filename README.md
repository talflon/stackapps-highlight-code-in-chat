UserScript posted by Nathan Osman [on stackapps](https://stackapps.com/q/3922/135137). As he said, it

> highlights `code` elements in chat messages so that they stand out more on the page.

I updated the `@include` URLs to match the ones currently used by Stack Overflow/Exchange chat:
- `http` → `https`
- `chat.meta.stackoverflow.com` → `chat.meta.stackexchange.com`

Here's a quick [link to install](https://github.com/talflon/stackapps-highlight-code-in-chat/raw/refs/heads/main/code_highlight.user.js).
